# Frontend - Ứng Dụng Đặt Thức Ăn Nhanh

Thư mục này chứa toàn bộ source code giao diện ứng dụng di động dành cho khách hàng và cửa hàng, sử dụng mô hình gọi API từ Backend.

## Công nghệ sử dụng
*   **Framework:** React Native (Expo)
*   **Navigation:** React Navigation
*   **State Management:** Redux Toolkit / Context API
*   **HTTP Client:** Axios (gọi API Backend)

## Phân công thực hiện
*   **Người phụ trách:** Trịnh Nhật Hoàng
*   **Nhánh Git:** `frontend`

## Cách khởi chạy dự án (phác thảo)
1. Di chuyển vào thư mục frontend: `cd frontend`
2. Cài đặt thư viện: `npm install`
3. Chạy ứng dụng bằng Expo: `npx expo start`
4. Quét mã QR bằng ứng dụng Expo Go trên điện thoại để trải nghiệm.
